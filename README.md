## Introduction
py-socket-server is a high-performance/low-latency/open-source Game Server developed based on Python.  

## Installation
run directly

```
python .\setup.py install
```

## Features
* WS/WSS Push/Play
* XMLS Push/Play
* Extended logging

## Roadmap
* HTTP-API
* Xarium client support
